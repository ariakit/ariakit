`Block` is [Box](../Box/Box.md) with `display: block`. By default it renders as a `<div>`.

```jsx
import { Block } from "reakit";

<React.Fragment>
  <Block
    width="100px"
    height="100px"
    backgroundColor="rgb(219, 112, 147)"
  />
  <Block
    width="100px"
    height="100px"
    backgroundColor="rgb(219, 112, 198)"
  />
  <Block
    width="100px"
    height="100px"
    backgroundColor="rgb(205, 112, 219)"
  />
</React.Fragment>
```
