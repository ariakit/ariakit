Solid `Button`, `Command`, and `Focusable` components

The [`Button`](https://solid.ariakit.com/reference/button), [`Command`](https://solid.ariakit.com/reference/command), and [`Focusable`](https://solid.ariakit.com/reference/focusable) components are now available in `@ariakit/solid`.

```tsx
import { Button } from "@ariakit/solid";

<Button>Button</Button>;
```
