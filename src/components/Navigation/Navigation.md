`Navigation` renders a `<nav>` by default.

```jsx
import { List } from "reakit";

<Navigation>
  <List>
    <List.Item>
      <Link href="https://github.com/reakit/reakit">GitHub</Link>
    </List.Item>
    <List.Item>
      <Link href="https://google.com">Google</Link>
    </List.Item>
  </List>
</Navigation>
```
