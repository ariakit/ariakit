`Table.Wrapper` is composed from [Base](/components/primitives/base) . Table.Wrapper  renders by default as a `<div />`.

- `Table.Wrapper`

```jsx
import { TableWrapper } from "reakit";
 <Table.Wrapper>
    <Table>
      <Table.Caption>A test table with merged cells</Table.Caption>
      ...
    </Table>
  </Table.Wrapper>
```
