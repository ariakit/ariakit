# Tab

<div data-description>

Display one panel of content at a time with a tabbed interface. This component is based on the [WAI-ARIA Tabs Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabpanel/).

</div>

## API

```jsx

useTabContext()

  <TabList>
    <Tab />
  </TabList>
  <TabPanel />
</TabProvider>
```
