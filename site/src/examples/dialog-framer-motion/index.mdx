## `AnimatePresence`

We use the [`AnimatePresence`](https://motion.dev/docs/react-animate-presence) component from Framer Motion to animate the Ariakit [Dialog](/components/dialog/) component when it gets mounted and unmounted from the DOM.

```jsx
<AnimatePresence>
  {mounted && (
    <Ariakit.Dialog
      store={dialog}
      alwaysVisible
      backdrop={
        <motion.div
          initial={{ opacity: 0 }}
          animate={{ opacity: 1 }}
          exit={{ opacity: 0 }}
        />
      }
      render={
        <motion.div
          initial={{ opacity: 0, scale: 0.95 }}
          animate={{ opacity: 1, scale: 1 }}
          exit={{ opacity: 0, scale: 0.95 }}
        />
      }
    />
  )}
</AnimatePresence>
```

To dynamically render the dialog component, you can use the [`mounted`](/react/components/dialog/use-dialog-store/#state-mounted) state that can be retrieved from the store:

```jsx
const dialog = Ariakit.useDialogStore();
const mounted = Ariakit.useStoreState(dialog, "mounted");
```

You can learn more about reading state from the store on the [Component stores](/guide/component-stores/#reading-the-state) guide.
