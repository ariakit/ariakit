Click a button to either [`show`](/react/components/disclosure/use-disclosure-store/#return-prop-show) (expand, open) or [`hide`](/react/components/disclosure/use-disclosure-store/#return-prop-hide) (collapse, close) a collapsible content element in React. This component is based on the [WAI-ARIA Disclosure Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/).
